JasperSample
======================
Jasperを用いた帳票出力のサンプルです。  

使い方
------
１．JasperSampleフォルダを、C:\wfdev\workspace 直下に配置する。  
２．Eclipseに既存プロジェクトとしてJasperSampleをインポートする。  
３．プロジェクトをビルドする。  
４．Sample1を実行する。  
５．C:\wfdev\workspace\JasperSample\template\report1.jrxmlを読み込み、同階層にreport1.pdfが出力される。
